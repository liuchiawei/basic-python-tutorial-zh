## 各章節詳細說明

### 第 1 章：認識 Python `ch-01-intro-python`

**本章重點：**

- 了解 Python 是什麼、其設計哲學與特性
- 認識 Python 的適用場景（網頁開發、資料分析、自動化、AI/ML 等）
- 理解為什麼要學 Python 以及與其他語言的比較

**學習目標：**

- [ ] 能說明 Python 的主要特點（簡潔、易讀、多用途）
- [ ] 能列舉至少 3 種 Python 的應用領域
- [ ] 能簡單比較 Python 與 C、Java、JavaScript 的差異

**適合對象：** 完全零基礎的初學者

---

### 第 2 章：環境安裝與設定 `ch-02-setup-environment`

**本章重點：**

- 安裝 Python 直譯器（官方發行版或 Anaconda）
- 選擇編輯器或 IDE（VS Code、PyCharm、Sublime 等）
- 驗證安裝是否成功，確認版本與路徑

**學習目標：**

- [ ] 能正確安裝 Python 3.x 並執行 `python --version`
- [ ] 能選擇並設定適合的開發環境
- [ ] 能使用終端機或命令提示字元執行 Python

**前置需求：** 無  
**實作建議：** 實際在電腦上完成安裝，並截圖記錄環境資訊

---

### 第 3 章：第一個程式 — Hello World `ch-03-hello-world`

**本章重點：**

- 撰寫與執行第一個 Python 程式
- 認識程式的基本結構（註解、程式碼區塊）
- 理解 `.py` 檔案的建立與執行流程

**學習目標：**

- [ ] 能建立 `.py` 檔案並執行
- [ ] 能寫出 `print("Hello, World!")` 並看到輸出
- [ ] 能使用註解 `#` 說明程式碼

**前置需求：** 第 2 章（環境已安裝）  
**實作建議：** 建立 `hello.py`，嘗試修改輸出的文字

---

### 第 4 章：變數與資料型別 `ch-04-variables-data-types`

**本章重點：**

- 變數的命名規則與良好慣例
- 基本資料型別：整數（`int`）、浮點數（`float`）、布林（`bool`）、字串（`str`）
- 型別轉換與 `type()` 函式

**學習目標：**

- [ ] 能宣告變數並賦值，了解變數可重複賦值
- [ ] 能辨識並使用 `int`、`float`、`bool`、`str`
- [ ] 能使用 `type()` 檢查型別，並進行基本型別轉換

**關鍵概念：**

- 動態型別：不需事先宣告型別
- 合法變數名稱：字母、數字、底線，不可數字開頭

---

### 第 5 章：運算子 `ch-05-operators`

**本章重點：**

- 算術運算子：`+`、`-`、`*`、`/`、`//`、`%`、`**`
- 比較運算子：`==`、`!=`、`>`、`<`、`>=`、`<=`
- 邏輯運算子：`and`、`or`、`not`
- 運算子的優先順序與括號的使用

**學習目標：**

- [ ] 能正確使用算術、比較、邏輯運算子
- [ ] 能理解整數除法 `//` 與取餘數 `%` 的差異
- [ ] 能計算複雜表達式的結果（考慮優先順序）

**前置需求：** 第 4 章（變數與資料型別）

---

### 第 6 章：字串 `ch-06-strings`

**本章重點：**

- 字串的建立（單引號、雙引號、三引號）
- 索引（index）與切片（slicing）：`s[0]`、`s[1:4]`
- 常用方法：`upper()`、`lower()`、`strip()`、`split()`、`join()`
- 格式化輸出：f-string、`format()`、`%`

**學習目標：**

- [ ] 能使用索引與切片存取字串內容
- [ ] 能使用常見字串方法進行處理
- [ ] 能使用 f-string 或其他方式格式化輸出

**關鍵概念：**

- 字串為不可變（immutable）
- 負索引：`-1` 表示最後一個字元

---

### 第 7 章：輸入與輸出 `ch-07-input-output`

**本章重點：**

- 使用 `print()` 輸出：預設換行、`end`、`sep` 參數
- 使用 `input()` 接收使用者輸入
- 將 `input()` 的結果轉換為數值型別
- 基本格式化與輸出排版

**學習目標：**

- [ ] 能使用 `print()` 控制輸出格式
- [ ] 能使用 `input()` 取得使用者輸入
- [ ] 能正確處理 `input()` 回傳的字串並轉型

**前置需求：** 第 4、5、6 章  
**實作建議：** 寫一個簡單的互動程式（如：詢問姓名並打招呼）

---

### 第 8 章：條件判斷 `ch-08-conditionals`

**本章重點：**

- `if`、`elif`、`else` 語法
- 條件表達式的撰寫（使用比較與邏輯運算子）
- 巢狀條件判斷的撰寫方式與縮排規則

**學習目標：**

- [ ] 能根據條件執行不同程式碼區塊
- [ ] 能正確使用 `elif` 處理多分支
- [ ] 能撰寫巢狀 `if` 並維持正確縮排

**關鍵概念：**

- Python 使用縮排（通常 4 空格）定義程式區塊
- 每個 `if` 區塊可包含多行程式碼

---

### 第 9 章：迴圈 `ch-09-loops`

**本章重點：**

- `for` 迴圈與 `range()` 的用法
- `while` 迴圈與終止條件
- `break` 跳出迴圈、`continue` 略過本次迭代
- 巢狀迴圈的基本概念

**學習目標：**

- [ ] 能使用 `for` 搭配 `range()` 重複執行
- [ ] 能使用 `while` 處理條件式重複
- [ ] 能適時使用 `break`、`continue` 控制流程

**前置需求：** 第 8 章（條件判斷）  
**實作建議：** 練習九九乘法表、猜數字遊戲等

---

### 第 10 章：資料結構 `ch-10-data-structures`

**本章重點：**

- 列表（list）：建立、索引、切片、常用方法（`append`、`insert`、`remove`）
- 元組（tuple）：建立、不可變特性、與 list 的差異
- 字典（dict）：鍵值對、存取、常用方法
- 集合（set）：不重複、常用方法（`add`、`remove`）

**學習目標：**

- [ ] 能建立並操作 list、tuple、dict、set
- [ ] 能根據情境選擇合適的資料結構
- [ ] 能理解可變（mutable）與不可變（immutable）的差異

**關鍵概念：**

- list：有序、可變、可重複
- tuple：有序、不可變
- dict：鍵值對、鍵唯一
- set：無序、不重複

---

### 第 11 章：函式 `ch-11-functions`

**本章重點：**

- 定義函式：`def` 語法
- 參數與回傳值：`return` 的使用
- 預設參數、關鍵字參數、可變參數（`*args`、`**kwargs`）
- 函式的作用域（scope）概念

**學習目標：**

- [ ] 能定義並呼叫自訂函式
- [ ] 能使用參數與回傳值傳遞資料
- [ ] 能使用預設參數與關鍵字參數

**前置需求：** 第 10 章（資料結構）  
**實作建議：** 將之前的程式邏輯重構為函式

---

### 第 12 章：模組與套件 `ch-12-modules-packages`

**本章重點：**

- 使用 `import` 載入模組
- 認識常用內建模組：`math`、`random`、`datetime`
- 套件的概念與 `__init__.py`
- 使用 `from ... import ...` 的寫法

**學習目標：**

- [ ] 能使用 `import` 載入並使用模組
- [ ] 能查詢並使用內建模組的函式與常數
- [ ] 能理解套件結構並建立簡單的自訂套件

**前置需求：** 第 11 章（函式）  
**銜接：** 完成本章後可進入進階篇（檔案操作、例外處理等）

## 學習建議

1. **循序漸進**：建議依章節順序學習，每章概念大多建立在前章之上。
2. **動手實作**：閱讀後務必自己寫程式練習，可依各章「實作建議」進行。
3. **善用資源**：遇到問題可查官方文件、Stack Overflow 或 Python 社群。
4. **完成基礎篇後**：你將具備撰寫小型腳本、處理基本資料與邏輯的能力，可繼續學習進階篇。
