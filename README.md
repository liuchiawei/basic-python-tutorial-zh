# Python 教學：從基礎到進階

> 一套完整的 Python 程式語言學習教材，涵蓋基礎、進階與高級主題，適合零基礎初學者循序漸進學習。

---

## 📖 簡介

本專案提供結構化的 Python 教學內容，從認識 Python、環境安裝，到變數、運算子、字串、條件判斷、迴圈、資料結構、函式、模組與套件等核心概念，並延伸至檔案操作、物件導向、非同步程式設計等高級主題。

## 🚀 快速開始

### 環境需求

- Python 3.x
- 建議使用 VS Code、PyCharm 或其他支援 Python 的 IDE

### 執行範例

```bash
# 進入 Basic 目錄
cd Basic

# 執行 Hello World
python 03-hello-world.py

# 執行變數與資料型別範例
python 04-variables-data-types.py
```

## 📝 學習建議

1. **循序漸進**：依章節順序學習，每章概念建立在前章之上
2. **動手實作**：閱讀後務必自己寫程式練習
3. **善用資源**：可查官方文件、Stack Overflow 或 Python 社群
4. **完成基礎篇後**：將具備撰寫小型腳本、處理基本資料與邏輯的能力
