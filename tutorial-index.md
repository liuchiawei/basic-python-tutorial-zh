# Python 教學大綱：從基礎到進階

---

## 基礎篇

### 第 1 章：認識 Python `ch-01-intro-python`

Python 是什麼、適用場景、為什麼要學 Python，以及與其他語言的比較。

### 第 2 章：環境安裝與設定 `ch-02-setup-environment`

安裝 Python 直譯器、選擇編輯器或 IDE、驗證安裝是否成功。

### 第 3 章：第一個程式 — Hello World `ch-03-hello-world`

撰寫與執行第一個 Python 程式，認識程式的基本結構。

### 第 4 章：變數與資料型別 `ch-04-variables-data-types`

變數的命名規則、基本資料型別（整數、浮點數、布林、字串）的介紹與使用。

### 第 5 章：運算子 `ch-05-operators`

算術運算子、比較運算子、邏輯運算子，以及運算子的優先順序。

### 第 6 章：字串 `ch-06-strings`

字串的建立、索引、切片、常用方法與格式化輸出。

### 第 7 章：輸入與輸出 `ch-07-input-output`

使用 `print()` 輸出、使用 `input()` 接收使用者輸入，以及基本格式化。

### 第 8 章：條件判斷 `ch-08-conditionals`

`if`、`elif`、`else` 語法，以及巢狀條件判斷的撰寫方式。

### 第 9 章：迴圈 `ch-09-loops`

`for` 迴圈與 `range()`、`while` 迴圈，以及 `break`、`continue` 的使用。

### 第 10 章：資料結構 `ch-10-data-structures`

列表（list）、元組（tuple）、字典（dict）、集合（set）的建立、操作與常見用法。

### 第 11 章：函式 `ch-11-functions`

定義函式、參數與回傳值、預設參數、關鍵字參數與可變參數。

### 第 12 章：模組與套件 `ch-12-modules-packages`

使用 `import` 載入模組、認識常用內建模組，以及套件的概念與使用方式。

---

## 進階篇

### 第 13 章：檔案與資料夾操作 `ch-13-file-operations`

讀寫文字檔、二進位檔、使用 `pathlib` 或 `os` 管理檔案與目錄。

### 第 14 章：例外處理 `ch-14-exception-handling`

使用 `try`、`except`、`finally` 處理錯誤，以及自訂例外類型。

### 第 15 章：物件導向程式設計 — 類別與物件 `ch-15-oop-classes-objects`

定義類別、建立物件、屬性（attribute）與方法（method）的基礎概念。

### 第 16 章：繼承與多型 `ch-16-inheritance-polymorphism`

父類別與子類別、覆寫方法、多型的概念與實作方式。

### 第 17 章：迭代器與生成器 `ch-17-iterators-generators`

`iter()`、`next()`、`yield` 的用法，以及生成器表達式。

### 第 18 章：裝飾器 `ch-18-decorators`

什麼是裝飾器、如何撰寫與使用裝飾器，以及常見使用情境。

### 第 19 章：正則表達式 `ch-19-regex`

使用 `re` 模組進行字串模式匹配、搜尋與取代。

---

## 高級篇

### 第 20 章：常用標準庫 `ch-20-standard-library`

認識 `collections`、`itertools`、`datetime`、`json` 等常用標準庫。

### 第 21 章：第三方套件管理 — pip `ch-21-pip-package-manager`

使用 `pip` 安裝、更新、移除套件，以及 `requirements.txt` 的用法。

### 第 22 章：虛擬環境 `ch-22-virtual-environments`

使用 `venv` 或 `virtualenv` 建立隔離的 Python 環境，避免套件衝突。

### 第 23 章：多執行緒與多進程 `ch-23-threading-multiprocessing`

`threading` 與 `multiprocessing` 模組，以及平行執行的基本概念。

### 第 24 章：非同步程式設計 `ch-24-async-programming`

`async`、`await` 語法，以及 `asyncio` 模組的入門使用。

### 第 25 章：測試與除錯 `ch-25-testing-debugging`

撰寫單元測試（`unittest`）、使用 `pytest`，以及常見除錯技巧。
